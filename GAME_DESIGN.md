# 天空之山 Web（SkyHill-like）— 游戏设计文档 / 规则

> 末世生存 Roguelike。灵感来自 SkyHill。纯 Web、零依赖、可在浏览器直接运行。
> 本文件是开发前的规则约定，作为后续实现的唯一依据。

---

## 1. 背景与目标

一场生化战争爆发。你在 **SkyHill 酒店 100 层顶层套房**中幸存。
为了逃生，你必须从 **100 层一路向下打到 1 层**，离开酒店。

- **胜利**：抵达 1 层并触发逃生出口。
- **失败（永久死亡）**：生命归零，或饥饿/口渴导致衰竭死亡。
- 每一局都是独立的 Roguelike 流程（Permadeath）。

---

## 2. 核心循环（Game Loop）

```
顶层基地(休整) ──→ 下楼探索 ──→ 房间事件(战斗/搜刮/特殊) ──→ 继续下楼
      ↑                                                        │
      └──────────────── 体力不足/受伤时返回基地 ────────────────┘
```

1. **基地阶段**（顶层套房）：睡觉恢复体力、做饭、合成、整理背包。
2. **探索阶段**：逐层向下，每层是一个房间，开门触发随机事件。
3. **每个动作消耗时间与体力**；时间推进会增加饥饿；饥饿过高会持续掉血/降低恢复。

---

## 3. 角色属性（Stats）

### 3.1 即时状态条
| 状态 | 说明 | 归零后果 |
|---|---|---|
| **生命 HP** | 受伤减少，治疗/休息恢复 | 死亡 |
| **体力 Stamina** | 探索/攻击/合成消耗，睡觉恢复 | 无法行动，被迫休息或受惩罚 |
| **饥饿 Hunger** | 随时间累积（数值越高越饿） | 满值后每回合掉 HP、恢复减半 |

### 3.2 角色能力值（开局选择职业/天赋决定初值）
| 能力 | 影响 |
|---|---|
| **力量 STR** | 近战伤害、负重上限 |
| **命中 ACC** | 攻击命中率 |
| **敏捷 DEX** | 闪避率、动作体力消耗 |
| **智力 INT** | 合成成功率、物品鉴定、解谜 |

### 3.3 开局职业（示例，初版 3 个）
- **保安**：高 STR/HP，起始近战武器（警棍）。
- **工程师**：高 INT，起始合成材料 + 配方，开局自带工具。
- **运动员**：高 DEX/体力，移动与逃跑消耗低。

---

## 4. 楼层与房间系统

- 楼层 100 → 1，每层一个「房间门」。
- 开门 → 按权重随机抽取 **房间类型**：
  | 房间 | 内容 |
  |---|---|
  | 空房 | 少量随机搜刮或一无所获 |
  | 物资房 | 食物 / 材料 / 弹药 / 药品 |
  | 怪物房 | 进入回合制战斗，胜利后可搜刮 |
  | 锁房 | 需钥匙或撬锁工具，开启后高价值奖励 |
  | 特殊房 | 商人 NPC、休息点、剧情碎片、稀有装备 |
- 楼层越低，怪物越强、稀有掉落越好（深度难度曲线）。
- 玩家可选择 **下楼**（前进）或 **返回基地**（回顶层休整，消耗时间）。

---

## 5. 时间 / 饥饿 / 休息

- 时间以「回合 / 天」推进；探索、战斗、合成、移动都会消耗时间。
- 每过一段时间饥饿上升；在基地 **做饭进食** 可降低饥饿。
- 在基地 **睡觉**：大幅恢复体力、少量恢复 HP，但推进时间（饥饿上升）。
- 饥饿满值：每回合掉 HP，且体力/HP 恢复效率减半。

---

## 6. 战斗系统（回合制）

每回合玩家从以下行动中选择其一：
- **攻击**：用当前武器。命中率 = f(ACC, 武器, 怪物闪避)；伤害 = 武器基础 + STR 加成。
- **使用物品**：吃药、用绷带、投掷物等。
- **防御**：减少本回合受伤，略微回体力。
- **逃跑**：成功率与 DEX 相关；失败则被怪物追击一回合。

怪物有自己的 HP / 攻击 / 闪避，随楼层深度成长，部分怪有特殊能力（毒、流血、护甲）。

---

## 7. 物品与背包

- **武器**：拳头（默认）、刀、铁管、手枪（需弹药）等。
- **护甲/防具**：减伤。
- **食物**：生食（饥饿恢复低，可能有副作用）/ 熟食（恢复高）。
- **药品**：医疗包、绷带、解毒剂。
- **材料**：废铁、化学剂（实现版按现有 CC0 素材精简为这两种核心材料）。
- **关键道具**：钥匙、撬棍、工具。
- 背包有 **格子/负重上限**（受 STR 影响），需要取舍。

---

## 8. 合成系统（Crafting）

- 基地有功能站点：
  - **工作台**：制作武器/工具（如 废铁 + 胶带 → 简易刀）。
  - **炉灶**：把生食做成熟食。
  - **床**：睡觉。
- 合成消耗材料，成功率受 INT 影响（失败损耗材料）。
- 实现版配方：
  | 产物 | 配方 | 站点 |
  |---|---|---|
  | 铁管（撬锁/武器） | 废铁 ×2 | 工作台 |
  | 尖刀 | 废铁 ×3 | 工作台 |
  | 防暴盾 | 废铁 ×4 | 工作台 |
  | 医疗包 | 化学剂 ×2 | 工作台 |
  | 热食 | 罐头食物 ×1 + 化学剂 ×1 | 炉灶 |
  | 能量饮料 | 化学剂 ×2 | 炉灶 |

## 9. 存档

- 使用 `localStorage` 自动保存当前局进度（永久死亡后清空该局）。
- 设置项（音量等）单独保存。

---

## 10. 技术方案

- **栈**：原生 JavaScript + HTML5 + CSS，**零依赖、无构建步骤**。
- **渲染**：DOM/CSS 负责 UI 与菜单；Canvas 负责战斗/场景表现。
- **架构**：状态机驱动（基地 / 探索 / 战斗 / 菜单），数据与逻辑分离，便于测试。
- **目录（初版）**：
  ```
  index.html
  /src        游戏逻辑（state、combat、items、floors…）
  /assets     图像、音效、字体
  /data       配置：怪物表、物品表、配方表、掉落表
  /tests      自动化测试 / 截图回归
  ```
- **测试**：本地静态服务器 + 无头 Chromium 自动截图与逻辑单测。

## 11. 美术 / 音效资源

- **第一版（占位）**：emoji + 纯 CSS 绘制，零下载。
- **正式版（全部免费 / CC0 可商用）**：
  - 图像：Kenney.nl（Roguelike/RPG/1-bit）、OpenGameArt。
  - 音效/BGM：Kenney Audio、Freesound（CC0）。
  - 字体：Google Fonts。
- **需要用户提供**：若需特定画风的角色立绘 / 专属美术，免费库无合适资源时，会列出清单交由用户提供。

## 12. 开发里程碑

- **M0 脚手架**：项目结构 + 主菜单 + 状态机 + 占位 UI。
- **M1 核心循环**：楼层推进、探索、状态条（HP/体力/饥饿）、基地睡觉。
- **M2 战斗**：回合制战斗、怪物表、掉落。
- **M3 物品/背包/合成**：物品系统、背包、炉灶/工作台合成。
- **M4 内容与平衡**：房间类型扩充、难度曲线、商人/特殊房、胜负结算。
- **M5 美化**：接入 CC0 美术与音效、动画、打磨。

> 每个里程碑结束我会跑测试 + 截图给你看，确认后再进入下一阶段。
